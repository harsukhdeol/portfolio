I used Dynamo to automate the creation of routine engineering drawings used in HVAC and electrical systems of all buildings. Dynamo is a visual programming language based off Python that uses Revit API to automate drafting and engineering calculations. Below is a sample of the tasks I automated with this program.

Place Air Terminals and VAVs

This script placed the correct VAV family according to the specified location and an array of air terminals spaced envenly in the room with the specified dimensions

Connecting Pipes and Valves to VAV

This script connects hot water supply and return pipes to the VAV in the specified direction and attaches the correct pipe accessories and valves.

Fire Alarm Riser

This script finds all the fire alarm devices in the model and organizes them by Level and HVAC zone to create a fire alarm drafting view

Wall Devices

This script hosts cable hangers, receptacles and pull stations to the selected wall face and spaces them according to Ontario Building Code (OBC)

Create Levels, Views and Sheets

This script creates a new level at the specified elevation, creates a floor plan on that level and plots that floor plan on a sheet. It allows the user to name the level, floor plan and sheet on the user interface.

Tag Family

This script searched for all the mechanical equipment in the model and sorts them according to type. Then it applies the correct tag family to each element by analyzing the category data.
